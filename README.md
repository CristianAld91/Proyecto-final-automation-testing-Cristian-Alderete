## 📌 Presentación General

Este proyecto reúne el trabajo realizado en las dos etapas de la cursada de **Automatización QA**:

1. **Etapa 1 – Automatización Web (Saucedemo con Selenium):**  
   Se implementaron pruebas automatizadas sobre la aplicación web de Sauce Labs, verificando funcionalidades clave como inicio de sesión, catálogo de productos, carrito de compras y menú lateral.  
   - Framework utilizado: **Selenium + pytest**  
   - Patrón aplicado: **Page Object Model (POM)**  
   - Reportes generados con **pytest-html**

2. **Etapa 2 – Automatización de API (JSONPlaceholder con requests):**  
   Se desarrollaron pruebas automatizadas sobre endpoints de la API pública JSONPlaceholder, organizadas con el patrón POM para mantener modularidad y reutilización.  
   - Framework utilizado: **pytest + pytest-check**  
   - Librerías: **requests, logging**  
   - Endpoints cubiertos: `/posts` y `/users`
